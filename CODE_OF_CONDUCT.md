# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and
welcoming experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and ask for clarification when needed.
- Accept responsibility for mistakes and correct them promptly.
- Focus criticism on ideas, code, and project outcomes.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks or sustained disruption.
- Publishing private information without explicit permission.
- Sexualized language or imagery in project spaces.

## Scope

This code of conduct applies in all project spaces, including issues, pull
requests, documentation, chat, events, and other community interactions related
to this project.

## Enforcement

Report concerns to the project maintainers through the repository issue tracker
or the contact channels published for this project. Maintainers will review
reports and take action that is appropriate to the circumstances, including
warnings, content removal, or limiting participation in project spaces.
